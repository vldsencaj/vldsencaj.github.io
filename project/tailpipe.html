TailPipe

Carbon emissions calculator for cloud computing.

Project Brief

With AppGlue, its core functionality (rules, flows, tasks, forms, etc) is unique and clearly has to be written by the team. There are other functions that can be done using standard libraries. These functions are critically important, but not different in such a way that we could not use standard libraries for this. The obvious ones include things like data storage (Raven and Mongo), authentication (SAML and JWT) and service communication (REST and Swagger).
